## Description
A hybrid automatic/manual control system for environmental monitoring.

- Components: Photoresistor (A0), Status LEDs (Green, Yellow, Red).

- Logic:

    - Manual Mode: Users set custom light thresholds via Serial commands (e.g., SET LOW 50).

    - Automatic Mode: The system uses hardcoded thresholds to classify the environment (Cloudy, Clear, Bright Sunlight) and activates the corresponding LED status indicator.
